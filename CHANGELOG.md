# Changelog

All notable changes to MoFish Tool are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project follows semantic versioning.

## [1.0.0] - 2026-05-15

### Added

- Initial Marketplace-ready release of MoFish Tool for IntelliJ IDEA 2025.3+.
- Added quote modules for stocks, market indexes, funds, cryptocurrencies, and foreign exchange rates.
- Added watchlists with searchable assets and configurable card/table views.
- Added portfolio holding configuration and profit calculation.
- Added price and percentage-change reminders through IDE notifications.
- Added automatic refresh settings, refresh windows, and status bar profit summaries.
- Added local settings persistence for watchlists, holdings, reminders, refresh preferences, and optional AI configuration.
